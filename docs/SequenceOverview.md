# `SequenceOverview` (component)



## Props

### `height`

Height of the SequenceOverview (in pixels), e.g. `50`

type: `number`
defaultValue: `50`


### `tileHeight`

Height of a tile in the OverviewBar, e.g. `5`

type: `number`
defaultValue: `5`


### `tileWidth`

Width of a tile in the OverviewBar, e.g. `5`

type: `number`
defaultValue: `5`
